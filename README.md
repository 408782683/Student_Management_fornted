# Student Management Frontend

基于 `api.yaml` 的 Vue 3 + Vite 示例前端，覆盖高校实习教务管理系统提供的全部接口（认证、管理员基础数据维护、教师端、学生端）。

## 本地运行

```bash
npm install
npm run dev
```

默认后端地址为 `http://localhost:8080`，可在页面顶部调整。

## 功能概览

- **认证与验证码**：获取验证码、按角色登录。
- **密码管理**：教师与学生修改密码。
- **管理员**：学院/专业/课程/计划/教师/学生/班级/教学任务的新增，课表上传，教师/学生删除，以及全量列表查询。
- **教师端**：教学任务查询、成绩列表、课表获取、成绩录入、花名册导出。
- **学生端**：成绩查询、课表获取。
